O curso "Introdução à Análise de Dados para Pesquisa no SUS" foi desenvolvido para apoiar profissionais do Sistema Único de Saúde (SUS), estudantes de graduação da área da saúde e demais interessados em compreender e aplicar conceitos fundamentais de análise de dados no contexto da saúde pública. Totalmente autoinstrucional, o curso apresenta os princípios básicos da modelagem estatística, o uso de software livre e as boas práticas de visualização de dados, oferecendo um caminho acessível e estruturado para o desenvolvimento de competências essenciais em um cenário cada vez mais orientado por evidências.
